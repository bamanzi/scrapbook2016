Evernote最开始是靠给苹果做图片文字识别起家的外包公司，在图片文字识别这方面还是很强悍的

大致原理：图片同步到服务器后，有专门的OCR图片识别服务器，将笔记中图片做OCR处理，然后将识别到的关键字，存入笔记的索引中，索引保存在笔记的.enex XML文件，最下面的recoIndex element。这个element是看不到的，不能复制粘贴，只能在搜索的被索引。

How Evernote's image Recognition workds?
ENEX file—a portable XML export format for Evernote notes—and jump to the bottom of the file, I’ll find the recoIndex element. Contained within recoIndex are a number ofitem nodes. Each item represents a rectangle Evernote’s OCR system believes to contain text.
