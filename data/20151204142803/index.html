LXDE/Openbox, Create Keyboard Shortcuts ⌨

Xah Lee
, 2015-03-14, …, 2015-11-26

Edit the file at ~/.config/openbox/lxde-rc.xml

For example, to make F9 key to switch to emacs, add:

<!-- set F9 to switch to emacs -->
<keybind key="F9">
    <action name="Execute">
        <command>wmctrl -xa emacs</command>
    </action>
</keybind>

You will need to install wmctrl for switching windows by name. See: Linux: wmctrl Add Keyboard Shortcuts to Switch to App ⌨

Here's more examples.

<!-- switch to Firefox -->
<keybind key="F8">
  <action name="Execute">
    <command>wmctrl -xa firefox</command>
  </action>
</keybind>

<!-- resize window -->
<keybind key="C-F1">
  <action name="Resize"/>
</keybind>

<!-- move window -->
<keybind key="C-F2">
  <action name="Move"/>
</keybind>

<keybind key="F1">
  <action name="Iconify"/>
</keybind>

<!-- prev tab -->
<keybind key="F11">
  <action name="Execute">
    <command>xvkbd -no-jump-pointer -xsendevent -text '\C\[ISO_Left_Tab]'</command>
  </action>
</keybind>

<!-- next tab -->
<keybind key="F12">
  <action name="Execute">
    <command>xvkbd -no-jump-pointer -xsendevent -text '\C\[Tab]'</command>
  </action>
</keybind>

<keybind key="F13">
  <action name="ToggleMaximize"/>
</keybind>

<!-- close window. Ctrl+w -->
<keybind key="F14">
  <action name="Execute">
    <command>xvkbd -no-jump-pointer -xsendevent -text '\Cw'</command>
  </action>
</keybind>

<!-- like alt+tab -->
<keybind key="F15">
  <action name="NextWindow">
    <dialog>yes</dialog>
    <interactive>yes</interactive>
    <raise>yes</raise>
  </action>
</keybind>

How to reload lxde key config file?

# reload lxde's config file at ~/.config/openbox/lxde-rc.xml
openbox-lxde --reconfigure

Key Syntax

Here's the modifier syntax.

  • S → ⇧ Shift key
  • C → Ctrl key
  • A → Alt key
  • W → ❖ Super key (the Microsoft Windows logo key)
  • M → ◆ Meta key
  • H → Hyper key

For example, if you want 【Alt+7】, use <keybind key="A-7">.

The key names are standard X11 “keysyms”. 〔➤ Linux: How to Find X11 Keycode/Keysym ⌨〕

Here's the common ones:

  • ⇞ Page △ → Prior
  • ⇟ Page ▽ → Next
  • ↑ arrow → Up
  • ↓ arrow → Down
  • → arrow → Right
  • ← arrow → Left
  • Return ↩ → Return
  • Space → space
  • Tab ↹ → Tab
  • ⌫ Backspace → Backspace
  • ⌦ Delete → Delete
  • Esc → Escape

For example, if you want 【Alt+↑】, use <keybind key="A-Up">.

Remove Keys

If you want to remove virtual desktop switching keys such as 【Ctrl+Alt+→】, which is used by emacs, delete the following.
