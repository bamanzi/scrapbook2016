Cheaper Intel Compute Stick with Ubuntu Linux preinstalled coming this week

After introducing a tiny Windows PC in April, Intel will roll out an Ubuntu version in the coming days.

When the Compute Stick first launched in April, Intel said an Ubuntu version would follow. To be clear, this is not Canonical taking Intel‚Äôs reference design and creating its own version. This is an Intel-branded device loaded with the popular Linux distribution.
