How do I get wicd to automatically show me the networks available when my desktop starts?

Add a .desktop file to your ~/.config/autostart directory (create one if not available) with 3 lines saying : 

[Desktop Entry] 

Type=Application

Exec=wicd-client -n

See http://wicd.sourceforge.net/moinmoin/FAQ

See https://wiki.archlinux.org/index.php/LXDE#Autostart_Programs

How to launch skype automatically when starting the computer?

Add the @skype line in the /etc/xdg/lxsession/LXDE/autostart or /etc/xdg/lxsession/Lubuntu/autostart file. Refer to LXSession for more information.
