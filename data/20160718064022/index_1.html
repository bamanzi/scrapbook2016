XDG-URL

xdgurl allows easy installation of themes, wallpapers, etc. on most desktop environments on any distro with simply clicking on the "Install" button.

Installation Guide

To install XDG-URL, get the package from here: https://www.linux-apps.com/p/1136805/

After xdgurl is installed, you can click any "Install" button on openDesktop.org. Only supported filetypes have an "Install" button.

Browser Setup

If you use Firefox, a program selection window opens the first time you click on an Install button: Choose "xdgurl" as custom URL handler and tick "remember the file association".

If you don't get the program selection window or you are using another browser, please set the xdgurl program as custom URL handler for xdg:// scheme in the browser's settings.
