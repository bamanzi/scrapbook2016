by Lux » 2007-01-08 11:42

You need to edit ~/GNUstep/Defaults/WMRootMenu and make its contents look like this:
Code: Select all
"menu.hook"

After that, WMaker's rootmenu should show the Debian menu like it used to.

* * * * *

If you want to modify WMaker's rootmenu while still keeping the automatically updated Debian menu, you can make the
Debian menu a submenu in WMaker's rootmenu. Execute the following commands as a normal user (NOT as root):

rm ~/GNUstep/Defaults/WMRootMenu
cp /usr/share/WindowMaker/plmenu ~/GNUstep/Library/WindowMaker/plmenu

Then, in WPrefs.app or wmakerconf add "External Submenu" to the WMaker rootmenu, point it to
/etc/X11/WindowMaker/menu.hook and name it "Debian menu".

There are also localized versions of "plmenu" in /usr/share/WindowMaker/, should you happen to prefer a non-English menu.
