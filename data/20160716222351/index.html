应用简介

黑客键盘Hacker's Keyboard提供一切你需要的按键，方便地在Android上输入快捷键组合、特殊符号，无论是Coding还是CLI都将更为效率。

感谢@千石彬的分享


分类标签：系统工具 输入法 特殊键 code ime

v1.37

- Removed READ_CONTACTS and RECORD_AUDIO permissions. New voice typing engine, disabled using contacts for completions.
- Added Meta/Command/Windows key for use with remote desktop apps such as bVNC.
- Bugfixes for modifier key handling.
- New layouts: ta (Tamil, தமிழ்), "Español (Latinoamérica)", hu (Hungarian, Magyar).
